# Subtitle Vocab Companion (Personal Study)

A lightweight browser extension that saves the word you click/select (e.g., while using Language Reactor) together with nearby subtitle/context text, for personal vocabulary review.

## Features
- Save clicked/selected word + nearby context text
- Local-only storage (chrome.storage.local)
- Export TSV (for Anki) / Print to PDF

## Install (Developer Mode)
1. Open Chrome/Edge extensions page
2. Enable Developer Mode
3. Load unpacked extension
4. Select this project folder

## Usage
- While watching subtitles, click/select a word (e.g., using Language Reactor dictionary lookup)
- The extension captures the word and saves it with nearby context
- Open the popup to view/export

## Privacy
- No network requests
- No data upload
- All data stays on your device (chrome.storage.local)
